# 📌 Inheritance & Polymorphism Lab (Java)

## 📖 Overview
This project demonstrates core **Object-Oriented Programming (OOP)** concepts in Java, focusing on **inheritance** and **polymorphism**. It provides practical examples of how classes interact through different inheritance relationships and how runtime behavior is achieved using method overriding.

## 🧠 Concepts Covered
- Inheritance (Single, Multilevel, Hierarchical)
- Method Overriding
- Polymorphism (Runtime)
- Use of `super` keyword
- Class relationships in Java OOP

## 🧪 Project Features
- Example of parent and child classes
- Demonstration of method overriding
- Runtime polymorphism behavior
- Clean and simple Java class structure
